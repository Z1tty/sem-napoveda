SEM › Implementace

E-commerce platformy

Seznam Event Measurement (SEM) můžete aktivovat přímo v administraci e-shopu pomocí nativního modulu. Celý proces je navržený tak, aby byl rychlý, přehledný a nevyžadoval žádné technické zásahy – stačí doplnit identifikátor a zapnout měření.

Přehled podporovaných platforem

Aktuální přehled e-shopových platforem s nativní podporou SEM najdete na o-seznam.cz/reklama/seznam-event-measurement/.

🖱️ Aktivace na pár kliků bez nutnosti programování
📦 Nativní modul připravený k okamžitému použití
⚡ Rychlé spuštění měření klíčových událostí

Obecný postup aktivace

  1. Přihlaste se do administrace e-shopu

    Otevřete administraci svého e-shopu a přihlaste se.

  2. Přejděte do sekce nastavení měření

    V administraci vyhledejte sekci, kde spravujete měření nebo marketingové nástroje.

  3. Vyberte modul Seznam Event Measurement

    V seznamu dostupných nástrojů otevřete modul pro Seznam Event Measurement.

  4. Vložte své SEM ID

    Pro aktivaci použijte SEM ID.

  5. Aktivujte měření

    Zapněte měření zaškrtnutím příslušného pole.

  6. Uložte nastavení

    Uložte změny. Tím je aktivace v administraci e-shopu dokončena.

Platforma není v seznamu?

Pokud vaše platforma nativní modul pro SEM zatím nepodporuje, použijte jednu z těchto alternativních metod:

🏷️ Google Tag Manager Pokud platforma podporuje GTM, nasaďte SEM přes GTM šablonu bez úpravy kódu webu. 🖥️ Přímý skript Pokud máte přístup do HTML šablon platformy, vložte SEM skript přímo.
SEM › Implementation

E-commerce platforms

You can activate Seznam Event Measurement (SEM) directly in your e-shop administration using the native module. The entire process is designed to be fast, straightforward, and require no technical expertise – just enter your identifier and enable measurement.

Supported platforms overview

For the current list of e-commerce platforms with native SEM support, visit o-seznam.cz/reklama/seznam-event-measurement/.

🖱️ Activation in a few clicks – no coding required
📦 Native module ready to use out of the box
⚡ Quickly start measuring key events

General activation steps

  1. Log in to your e-shop administration

    Open your e-shop administration and log in.

  2. Go to the measurement settings section

    In the administration, find the section where you manage tracking or marketing tools.

  3. Select the Seznam Event Measurement module

    In the list of available tools, open the Seznam Event Measurement module.

  4. Enter your SEM ID

    Use your SEM ID to activate the module.

  5. Enable measurement

    Turn on measurement by checking the relevant checkbox.

  6. Save the settings

    Save your changes. Activation in the e-shop administration is now complete.

Platform not in the list?

If your platform does not yet support a native SEM module, use one of these alternative methods:

🏷️ Google Tag Manager If your platform supports GTM, deploy SEM via the GTM template without modifying your website code. 🖥️ Basic script If you have access to the platform's HTML templates, embed the SEM script directly.
