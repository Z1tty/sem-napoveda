Reference událostí

Kompletní přehled všech podporovaných SEM událostí, jejich parametrů a kódových ukázek. Události se odesílají metodou SEM('track', 'NázevUdálosti', data).

Základní pravidla Měna musí být vždy CZK. Ostatní měny přepočítejte před odesláním. Hodnoty value a unit_price jsou vždy bez DPH. DPH zasílejte v samostatných polích (value_tax, unit_price_tax).

Přehled všech událostí

Událost Kdy odesílat Povinné Doporučené DRTG
PageView Každá načtená stránka – – ✓ (produkty)
ViewContent Detail produktu nebo kategorie – content_type, currency, value, value_tax, contents ✓ (produkty i kategorie)
AddToCart Přidání do košíku – currency, value, value_tax, contents –
AddToWishlist Přidání do oblíbených – currency, value, value_tax, contents –
InitiateCheckout Vstup do procesu placení – currency, value, value_tax, contents –
AddPaymentInfo Zadání platebních údajů – currency, value, value_tax, contents –
Purchase Dokončení objednávky order_id, currency, value, value_tax content_type, contents, review_email, delivery_type, delivery_price –
Search Vyhledávání na webu – search_string –
Lead Odeslání formuláře / kontakt – currency, value, value_tax –
CompleteRegistration Dokončení registrace – status, currency, value, value_tax –
Subscribe Přihlášení k předplatnému – currency, value, value_tax, predicted_ltv –
StartTrial Zahájení zkušební verze – currency, value, value_tax, predicted_ltv –

PageView

Odesílejte na každé stránce webu. Základní skript volá PageView automaticky. U SPA aplikací (React, Vue atd.) volejte událost manuálně při každé změně routy.

PageView se také využívá pro dynamický retargeting produktů – pokud k události přidáte content_type: 'product' a ID produktu v poli contents.

JavaScript 
// Základní PageView (automaticky voláno skriptem)
SEM('track', 'PageView');

// PageView s daty produktu (pro DRTG)
SEM('track', 'PageView', {
  content_type: 'product',
  currency: 'CZK',
  value: 33490,
  value_tax: 7033,
  contents: [{
    id: 'ABC12345',        // ITEM_ID z XML feedu
    content_name: 'iPhone 15 Pro Max',
    unit_price: 33490,
    unit_price_tax: 7033
  }]
});

ViewContent

Odesílejte při zobrazení stránky produktu nebo stránky kategorie. Klíčová událost pro dynamický retargeting – pomocí content_type rozlišujete, zda jde o produkt nebo kategorii.

ParametrTypPovinnostPříkladPopis
content_typestring Doporučený 'product' Typ obsahu: product pro detail produktu, product_group pro kategorii, page pro ostatní stránky
currencystring Doporučený 'CZK' Vždy CZK. Ostatní měny přepočítejte.
valuenumber Doporučený 33490 Celková hodnota v CZK bez DPH
value_taxnumber Doporučený 7033 Hodnota DPH v CZK
contentsarray Doporučený viz ukázka Pole produktů. Viz popis objektu Contents[] níže.
JavaScript 
SEM('track', 'ViewContent', {
  content_type: 'product',
  currency: 'CZK',
  value: 33490,
  value_tax: 7033,
  contents: [{
    id: 'ABC12345',          // musí odpovídat ITEM_ID v XML feedu
    content_name: 'iPhone 15 Pro Max',
    content_category: 'Mobily, chytré hodinky, tablety | Mobilní telefony',
    unit_price: 33490,      // cena bez DPH
    unit_price_tax: 7033  // DPH
  }]
});
JavaScript 
// Pro stránku kategorie – content_type musí být 'product_group'
SEM('track', 'ViewContent', {
  content_type: 'product_group',
  currency: 'CZK',
  value: 0,
  value_tax: 0,
  contents: [{
    content_category: 'Mobilní telefony'  // kategorie pro DRTG
  }]
});

AddToCart a AddToWishlist

Odesílejte při přidání produktu do košíku (AddToCart) nebo do oblíbených (AddToWishlist). Struktura dat je pro obě události identická.

ParametrTypPovinnostPříkladPopis
currencystringDoporučený'CZK'Vždy CZK
valuenumberDoporučený33490Celková hodnota bez DPH
value_taxnumberDoporučený7033DPH celkem
contentsarrayDoporučenýviz ukázkaPřidané produkty. Viz Contents[].
JavaScript 
SEM('track', 'AddToCart', {  // nebo 'AddToWishlist'
  currency: 'CZK',
  value: 33490,
  value_tax: 7033,
  contents: [{
    id: 'ABC12345',
    content_name: 'iPhone 15 Pro Max',
    content_category: 'telefony',
    quantity: 1,
    unit_price: 33490,
    unit_price_tax: 7033
  }]
});

InitiateCheckout a AddPaymentInfo

Odesílejte při zahájení procesu objednávky (InitiateCheckout) a při zadání platebních údajů (AddPaymentInfo). Umožňují měřit míru opuštění košíku.

JavaScript 
SEM('track', 'InitiateCheckout', {  // nebo 'AddPaymentInfo'
  currency: 'CZK',
  value: 33990,
  value_tax: 7138,
  contents: [
    {
      id: 'ABC12345',
      quantity: 1,
      content_name: 'iPhone 15 Pro Max',
      content_category: 'telefony',
      unit_price: 33490,
      unit_price_tax: 7033
    },
    {
      id: 'KRYT999',
      quantity: 1,
      content_name: 'Ochranný kryt',
      content_category: 'příslušenství',
      unit_price: 500,
      unit_price_tax: 105
    }
  ]
});

Purchase

Nejdůležitější e-commerce událost – odesílejte ihned po dokončení objednávky na stránce potvrzení. Obsahuje kompletní data objednávky včetně produktů, dopravy a platby.

review_email musí být nehashovaný Parametr review_email slouží pro odesílání dotazníku spokojenosti Zboží.cz. Na rozdíl od ostatních e-mailových údajů jej nehashujte. Zasílejte jej jen pokud uživatel nevyjádřil nesouhlas se zasíláním dotazníků.
ParametrTypPovinnostPříkladPopis
order_idstringPovinný'OBJ-987654'Unikátní číslo objednávky. Slouží k deduplikaci.
currencystringPovinný'CZK'Vždy CZK
valuenumberPovinný33990Celková cena produktů bez DPH
value_taxnumberPovinný7138Celkové DPH za produkty
content_typestringDoporučený'product'Vždy product u nákupu produktů
contentsarrayDoporučenýviz ukázkaZakoupené produkty. Viz Contents[].
review_emailstringDoporučený'jan@email.cz'⚠️ NEHASHOVAT. E-mail pro dotazník Zboží.cz.
delivery_typestringDoporučený'CESKA_POSTA'Způsob dopravy dle DELIVERY_ID z XML feedu
delivery_pricenumberDoporučený120Cena dopravy bez DPH v CZK
delivery_price_taxnumberDoporučený25.20DPH z dopravy
payment_typestringVolitelný'karta'Způsob platby – libovolný řetězec
other_costsnumberVolitelný-50Další náklady/slevy bez DPH. Slevy = záporné číslo.
other_costs_taxnumberVolitelný-10.5DPH z dalších nákladů/slev
JavaScript 
SEM('track', 'Purchase', {
  content_type: 'product',
  currency: 'CZK',
  order_id: 'OBJ-987654',
  review_email: 'zakaznik@email.cz',  // ⚠️ NEHASHOVAT
  value: 33990,          // celková cena produktů bez DPH
  value_tax: 7138,       // celkové DPH za produkty
  delivery_type: 'CESKA_POSTA',
  delivery_price: 120,
  delivery_price_tax: 25.20,
  payment_type: 'karta',
  contents: [
    {
      id: 'ABC12345',
      quantity: 1,
      content_name: 'iPhone 15 Pro Max',
      content_category: 'telefony',
      unit_price: 33490,
      unit_price_tax: 7033
    },
    {
      id: 'KRYT999',
      quantity: 1,
      content_name: 'Ochranný kryt',
      content_category: 'příslušenství',
      unit_price: 500,
      unit_price_tax: 105
    }
  ]
});

Search

Odesílejte při každém vyhledávání na webu. Umožňuje cílit retargeting na uživatele hledající konkrétní produkty.

JavaScript 
SEM('track', 'Search', {
  search_string: 'bezdrátová sluchátka'
});

Lead, CompleteRegistration, Subscribe, StartTrial

Události pro non-e-commerce weby – lead generation, registrace, předplatné.

JavaScript 
SEM('track', 'Lead', {
  currency: 'CZK',
  value: 500,      // odhadovaná hodnota leadu
  value_tax: 105
});
JavaScript 
SEM('track', 'CompleteRegistration', {
  status: true,    // true = úspěšná registrace
  currency: 'CZK',
  value: 0,
  value_tax: 0
});
JavaScript 
SEM('track', 'Subscribe', {  // nebo 'StartTrial'
  currency: 'CZK',
  value: 199,           // cena předplatného (bez DPH)
  value_tax: 41.79,
  predicted_ltv: 2400  // odhadovaná celoživotní hodnota zákazníka
});

Ostatní události

Tyto události nemají povinné parametry a slouží pro měření specifických interakcí.

UdálostKdy použít
ContactUživatel kontaktoval firmu (telefon, chat, e-mail)
CustomizeProductUživatel přizpůsobil produkt (barva, konfigurace)
DonateDar na charitativní účel
ScheduleRezervace termínu (kadeřnictví, lékař...)
SubmitApplicationOdeslání žádosti nebo přihlášky
JavaScript 
// Bez parametrů
SEM('track', 'Contact');

// S volitelnou hodnotou
SEM('track', 'Schedule', {
  currency: 'CZK',
  value: 1000,
  value_tax: 210
});

Objekt produktu (contents[])

Pole contents se předává v událostech ViewContent, AddToCart, AddToWishlist, InitiateCheckout, AddPaymentInfo a Purchase. Každý objekt v poli popisuje jeden produkt v transakci.

ParametrTypDoporučenostPříklad hodnotyPopis
idstring Doporučený 'ABC12345' ID produktu – musí přesně odpovídat hodnotě ITEM_ID v XML feedu (Zboží.cz). Nutné pro dynamický retargeting.
quantitynumber Doporučený 2 Počet kusů daného produktu v košíku / objednávce.
unit_pricenumber Doporučený 33490 Cena za jeden kus v CZK bez DPH.
unit_price_taxnumber Doporučený 7033 Výše DPH za jeden kus v CZK.
content_namestring Volitelný 'iPhone 15 Pro Max' Název produktu. Lze použít jako podmínku retargetingu.
content_categorystring Volitelný 'Mobily | Mobilní telefony' Kategorie produktu. Lze použít jako podmínku retargetingu. Podporuje celý hierarchický řetězec z XML feedu, např. 'Příslušenství | Kryty | Apple' – celý řetězec pak lze použít jako přesnou podmínku.
Příklad – contents[] 
contents: [
  {
    id: 'ABC12345',               // ITEM_ID z XML feedu
    quantity: 1,
    unit_price: 33490,            // bez DPH
    unit_price_tax: 7033,         // pouze DPH
    content_name: 'iPhone 15 Pro Max',
    content_category: 'Mobilní telefony'
  },
  {
    id: 'KRYT999',
    quantity: 2,
    unit_price: 500,
    unit_price_tax: 105,
    content_name: 'Ochranný kryt'
  }
]
Hodnota value vs. součet contents Parametr value na úrovni události je celková hodnota transakce bez DPH a nemusí nutně odpovídat součtu unit_price × quantity ze všech položek – může zahrnovat slevy nebo jiné úpravy. Detailní rozpad je v contents[].
