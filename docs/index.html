Sklik Event Measurement (SEM)

Nový systém měření událostí, který nahrazuje staré retargetingové a konverzní kódy. SEM sjednocuje sběr dat pro kampaně, retargeting i optimalizaci v jednom skriptu.

Co je SEM a proč přecházet?

SEM (Sklik Event Measurement) je moderní systém měření událostí na vašem webu, který nahrazuje dosavadní retargetingové kódy a měřicí skripty konverzí pro Sklik i Seznam Nákupy. Místo několika oddělených skriptů stačí jeden SEM script sul.js.

SEM sbírá data o chování uživatelů (zobrazení produktů, přidání do košíku, nákupy atd.) a odesílá je do Skliku, kde je využíváte pro:

  • Měření konverzí – víte, které kampaně přinášejí objednávky, registrace nebo leady
  • Retargeting – oslovujete uživatele, kteří navštívili konkrétní stránky, produkty, kategorie nebo splnili definované podmínky událostí podle vaší definice
  • Dynamický retargeting (DRTG) – automaticky zobrazujete reklamy na konkrétní prohlížené produkty
  • Optimalizace kampaní – systém využívá nasbíraná data k lepšímu cílení

Jak začít – zvolte způsob implementace

Vyberte metodu, která odpovídá vašemu technickému řešení:

🖥️ Přímý skript Vložíte kód přímo do HTML šablony webu. Vhodné pro vývojáře s přístupem ke zdrojovému kódu. 🏷️ Google Tag Manager Nasadíte SEM přes GTM šablonu bez zásahu do kódu webu. Nejrychlejší cesta pro většinu inzerentů. 🛒 E-commerce platforma Mnoho e-commerce platforem nabízí hotový SEM doplněk – stačí zadat SEM ID. ⚙️ Server-to-Server Pokročilé měření z vašeho serveru přímo na Sklik endpoint. Pro vývojáře s vlastním backendem.

Sklik Event Measurement (SEM)

A new event measurement system replacing the legacy retargeting and conversion codes. SEM unifies data collection for campaigns, retargeting and optimization in a single script.

What is SEM and why migrate?

SEM (Sklik Event Measurement) is a modern event measurement system for your website that replaces the existing retargeting codes and conversion tracking scripts for both Sklik and Seznam Shopping. Instead of multiple separate scripts, a single SEM script sul.js is all you need.

SEM collects data about user behaviour (product views, add-to-cart actions, purchases, etc.) and sends it to Sklik, where you can use it for:

  • Conversion tracking – find out which campaigns drive orders, registrations or leads
  • Retargeting – reach users who visited specific pages, products or categories, or who met the event conditions defined in Event Management
  • Dynamic retargeting (DRTG) – automatically show ads for the exact products a user browsed
  • Campaign optimisation – the system uses collected data for smarter targeting

Getting started – choose your implementation method

Select the method that best fits your technical setup:

🖥️ Direct script Paste the code directly into your website's HTML template. Ideal for developers with source code access. 🏷️ Google Tag Manager Deploy SEM via a GTM template without touching your site's code. The fastest path for most advertisers. 🛒 E-commerce platform Many e-commerce platforms offer a ready-made SEM add-on – just enter your SEM ID. ⚙️ Server-to-Server Advanced server-side measurement sent directly to the Sklik endpoint. For developers with a custom backend.
