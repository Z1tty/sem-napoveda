Jak nasadit SEM

Průvodce výběrem správné metody implementace podle vašeho technického zázemí. SEM lze nasadit čtyřmi způsoby – vyberte ten, který nejlépe odpovídá vašemu webu.

Zvolte metodu implementace

Před nasazením SEM si rozmyslete, jak spravujete váš web a jaké nástroje máte k dispozici. Všechny čtyři metody jsou rovnocenné z pohledu nasbíraných dat – liší se pouze způsobem nasazení.

Metoda Kdo ji použije Přístup ke kódu webu Technická náročnost
Přímý skript Vývojář s přístupem do HTML šablon Vyžadován Střední – nutné přidat kód na každou stránku
Google Tag Manager Marketér nebo vývojář s GTM Není potřeba (jen GTM) Nízká – šablona v GTM galerii, bez kódování
E-commerce platforma Provozovatel Shoptet, Upgates a podobně Není potřeba (doplněk platformy) Velmi nízká – stačí zadat SEM ID
Server-to-Server (S2S) Vývojář s vlastním backendem Vyžadován (BE + FE) Vysoká – HTTP volání z vašeho serveru

Před nasazením: získejte SEM ID

Každý způsob implementace potřebuje SEM ID vašeho Sklik účtu. Najdete ho v:

  1. Přihlaste se do Skliku
  2. Klikněte na Nastavení účtu → Správa měření
  3. V tabulce uvidíte záznamy s SEM ID
E-shopy napojené na Zboží.cz Pokud máte provozovny Zboží.cz, každá má vlastní kombinované SEM ID (účet + provozovna). Pro Purchase konverze vždy použijte kombinované SEM ID příslušné provozovny, ne obecné SEM ID účtu.

Dva typy SEM ID v Správě měření

IdentifikátorTlačítkoKde ho použijete
SEM ID Zobrazit SEM kód · Zkopírovat Přímý skript → použijte „Zobrazit SEM kód" a zkopírujte celý <script> tag
GTM šablona / plugin → použijte „Zkopírovat" (samotné ID)
SEM ID pro S2S Zkopírovat Server-to-Server měření – parametr sem_id v payloadu

SEM ID pro S2S je viditelné po zapnutí přepínače Zobrazit pokročilé nastavení vpravo nahoře v sekci Správa měření.

Podrobnosti ke každé metodě

🖥️ Přímý skript Vložíte kód přímo do HTML šablony webu. Vhodné pro weby bez e-commerce nebo pro vývojáře s přístupem ke zdrojovému kódu. 🏷️ Google Tag Manager Nasadíte SEM přes GTM šablonu bez zásahu do kódu webu. Nejrychlejší cesta pro inzerenty s e-shopem a GTM. 🛒 E-commerce platforma Shoptet, Upgates a další nabízejí hotový doplněk. Stačí zadat SEM ID do nastavení pluginu. ⚙️ Server-to-Server Pokročilé měření přímo z vašeho serveru. Vyžaduje vlastní backend a FE skript pro cookies.

Po nasazení: otestujte a přepněte

  1. Ověřte implementaci v Sandboxu

    Otevřete testovací nástroj a zkontrolujte, zda přijímáte správné události s platnými parametry.

  2. Nastavte konverze a retargetingové seznamy

    V Správě měření vytvořte konverzní akce (Purchase, Lead…) a retargetingová publika před přepnutím.

  3. Přepněte na SEM

    V Skliku klikněte na Začít používat SEM. Od tohoto okamžiku se počítají pouze nová data. Přepnutí nelze vrátit zpět.

Paralelní provoz Během testování můžete provozovat starý retargetingový kód i SEM zároveň. Data se sbírají nezávisle – nedochází k duplicitám v účtování konverzí. Přepnutí provedete, až jste s implementací spokojeni.
